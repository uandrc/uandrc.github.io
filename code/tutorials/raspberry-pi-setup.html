How to Set Up a Raspberry Pi

A Raspberry Pi is a small computer. Here is how to start using one.

For simplicity, this tutorial assumes you are using a Raspberry Pi 2 Model B or above (among the most common models). Setup for other Raspberry Pis varies slightly, but generally follow the steps listed below.

You will need:

  • Raspberry Pi (wow!)
  • Unused (formatted) micro SD card*
  • USB-C or Micro USB Power supply (dependent on Raspberry Pi model)
  • HDMI or micro HDMI cable (dependent on Raspberry Pi model)
  • Mouse
  • Keyboard (not needed for the Raspberry Pi 400, which is built inside a keyboard)
  • Monitor
  • Monitor power supply
*Important: The micro SD card will be overwritten in the OS-installation process. Ensure there is absolutely nothing you will miss on the card before proceeding!

Instructions:

  1. Install and run Raspberry Pi Imager.
  2. Insert your micro SD card into your computer.
  3. If you are not familiar with Linux, select "Raspberry Pi OS (64-bit)" as your chosen OS. Otherwise, select an OS of your choosing*.
  4. Remove the micro SD card from your computer and insert it into your Raspberry Pi.
  5. Plug one end of the HDMI cable into the Raspberry Pi and the other end into the monitor.*
  6. Plug the mouse and keyboard into the Raspberry Pi.
  7. Plug the monitor into its power supply.
  8. Plug the Raspberry Pi power supply into an outlet.
  9. Turn the monitor on, then connect the Raspberry Pi with its power supply.
  10. Wait for the Raspberry Pi to boot.
*Note: If your Raspberry Pi has two HDMI ports, plug your main monitor into the HDMI0 port. Note: Raspberry Pi Imager allows you to select a custom .img file for your Pi's OS. This enables you to install Windows 11 and other OS on your Raspberry Pi.

Turning a Raspberry Pi off

Turning a Raspberry Pi on is as simple as pie: you plug it in. However, turning a Raspberry Pi off is not as simple as unplugging it while it is on. In fact, doing so can cause the micro SD card to corrupt and break your OS.

To properly turn your Raspberry Pi off:

  1. Open a terminal window.
  2. Enter the following command: sudo shutdown -h now
  3. Wait for the screen to go completely black (this may take several seconds, be patient!)
  4. Unplug your Raspberry Pi from its power source.
  5. Turn off your monitor.
