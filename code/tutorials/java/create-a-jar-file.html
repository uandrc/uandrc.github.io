How to Create a JAR file

A JAR (Java Archive) file enables you to bundle several Java assets (classes, image/sound resources, etc.) into a single file. It is very useful for app distribution.

Here is how to make a .jar file in IntelliJ.

Instructions:

Setup

  1. Verify your Java project compiles and runs without errors.
  2. Verify your Java project uses a JDK: Go to File -> Project Structure -> SDK. Ensure a Java SDK is selected.

Create a JAR file

  1. In IntelliJ, go to File -> Project Structure.
  2. Select Artifacts from the left sidebar.
  3. Click the + button.
  4. Choose JAR -> From modules with dependencies.
  5. In the window that appears, select the Main class (i.e., the class containing the main method).
  6. Click OK.
  7. Click OK in the Project Structure window.
  8. From the IntelliJ menu bar, go to Build -> Build Artifacts -> Build
  9. Navigate to the out/artifacts directory in your project. Look for your-project-name.jar.
  10. Run your .jar file by right-clicking on it and selecting Run 'your-project-name.jar'.
