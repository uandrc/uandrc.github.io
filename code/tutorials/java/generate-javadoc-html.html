How to Generate Javadoc

Javadoc is a tool for generating API documentation for Java code. It allows you to create a detailed map of your code so other programmers can more quickly and easily understand it.

Here is how to generate Javadoc in IntelliJ.

Instructions:

Setup

  1. Verify your Java code has Javadoc comments for all classes, constructors, methods, etc.
  2. Verify your Java project uses a JDK: Go to File -> Project Structure -> SDK. Ensure a Java SDK is selected.

Generate Javadoc HTML

  1. Create a folder you want to store the generated Javadoc inside.
  2. In IntelliJ, go to Tools -> Generate Javadoc.
  3. Specify your Javadoc scope and options (select the output directory you created in Step 1 here).
  4. Click Generate.
  5. By default, IntelliJ will automatically open the generated Javadoc in a browser window. If not, find and open the index.html file in your output directory to view the main Javadoc HTML file.
