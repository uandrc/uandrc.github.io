How to Convert a java file into an exe file

The average software user does not have Java installed, nor knows how to run .java files. Converting a .java file into the more recognized .exe file is important for app distribution.

Instructions:

Setup

  1. Create a JAR file of your Java project.
  2. Verify your JAR file runs without errors.

Create an .exe file

  1. Download and install Launch4j.
  2. Open Launch4j and select the Basic tab.
  3. Select the folder icon to the right of Output file. Choose the output directory your .exe will be created in. Specify the output file name, making sure your file name ends in .exe. Click Save.
  4. Select the folder icon to the right of Jar. Select your IntelliJ-generated JAR file.
  5. (Optional): Select an .ico file to customize the .exe icon. You can convert images to .ico here.
  6. Navigate to the JDK folder. On Windows File Explorer, the path to the JDK is C:\Program Files\Java. Copy the full directory path to the JDK, e.g., C:\Program Files\Java\jdk-23.
  7. In Launch4j, go to the JRE tab. Paste the path you copied in the previous step into the JRE paths field.
  8. Select the Gear icon near the top-left of the Launch4j window, (Build wrapper). Enter config in the file name field. Click Save.
  9. To run the generated .exe, navigate to the output directory and double-click the .exe file.
