Fun Java Facts

Things you might not know about the Java programming language.

  1. The first four bytes of every Java class file are always 0xCAFEBABE.
  2. Java has a mascot named Duke.
  3. Java was originally called “Oak,” named after a tree outside the office of its creator. The language was changed to "Java" after a trademark conflict.
  4. Before JDK 7, you could print "Hello World!" without the main method.
  5. Minecraft was originally written in Java.
  6. Jakarta EE, a standard for building Java apps, is named after the largest city on the island of Java.
  7. Java is younger than Python.
  8. Microsoft tried to recreate Java, calling it "J++".
  9. The Java Development Kit (JDK), version 1.6, has a class named:
    InternalFrameInternalFrameTitlePaneInternalFrameTitlePaneMaximizeButtonWindowNotFocusedState
  10. Java’s Log4j framework was exploited in what many security experts consider the most severe vulnerability ever discovered.
